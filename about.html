Verify a Certificate Issue (Admin)

Project overview

What is TruCert?

TruCert lets organizations issue tamper-proof certificates on-chain so anyone can independently verify authenticity using a certificate ID or QR.

Why blockchain?

Certificates and their metadata are anchored by smart contracts and IPFS links, making records durable, transparent, and resistant to forgery.

Core benefits

  • Immutable and verifiable records
  • Public verification, restricted issuance
  • Lower cost, eco-friendly, audit-ready

How it works

Issue a certificate (Admin)

  1. Open Admin and connect MetaMask when prompted.
  2. Fill all fields: Name, Course, Description, and IPFS URI (uploaded asset/metadata link).
  3. Click Add Data to send the transaction; wait for confirmation.
  4. Note the returned Certificate ID to share with recipients.
Go to Admin

Verify a certificate (Public)

  1. Open Verify and enter the Certificate ID.
  2. Fetch on-chain details including issuer address, timestamp, and fields.
  3. Open the IPFS link to view the referenced asset/metadata.
Go to Verify

Data fields

  • Field 1: Name
  • Field 2: Course
  • Field 3: Description
  • IPFS URI: Link to the artifact/metadata

Technology

Smart contracts

A deployed contract stores entries, enforces allowed issuers, emits events on additions, and exposes read-only getters for public verification.

Storage

Certificates reference IPFS URIs for images or JSON metadata, keeping on-chain data minimal while preserving integrity via content addressing.

Frontend

HTML, CSS, and JavaScript integrate with Web3 via MetaMask to submit transactions for issuance and to call read methods for verification.

Tips & notes

Admins

  • Ensure the connected account is allowed to add data.
  • Always provide a valid IPFS URI pointing to the correct asset/metadata.
  • Share the Certificate ID securely with recipients.

Verifiers

  • Use the exact Certificate ID provided by the issuer.
  • Confirm the issuer address and timestamp match expectations.
  • Open the IPFS link to inspect the referenced artifact.
